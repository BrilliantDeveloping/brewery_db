# BreweryDB

A Ruby library for interfacing with the [BreweryDB][] API.

[brewerydb]: http://www.brewerydb.com/


## Installation

Add this line to your application's Gemfile:

    gem 'brewery_db'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install brewery_db
